## 📖 Sobre o projeto

O **FocusFlow** é uma aplicação web inspirada na técnica Pomodoro, desenvolvida para transformar sessões de estudo e trabalho em uma experiência mais agradável e motivadora.

> 🚧 O projeto ainda está em desenvolvimento e novas funcionalidades serão adicionadas.

---

## ✨ Funcionalidades

- 🍅 Cronômetro Pomodoro
- ⏸️ Pausar, continuar e resetar sessões
- ⚙️ Personalização do tempo de foco, descanso e quantidade de ciclos
- 🐱 Mascote "Nori"
- 📈 Barra de progresso do mascote
- ✅ Missões diárias
- 🎵 Seleção de ambiente sonoro
- 🔊 Controle de volume
- 💬 Mensagens motivacionais durante a sessão

## 📷 Interface

O layout foi pensado para concentrar todas as funcionalidades em um único painel, facilitando o acompanhamento da sessão de estudos.

- Timer principal
- Mascote Nori
- Missões do dia
- Ambientes sonoros
- Personalização da sessão
